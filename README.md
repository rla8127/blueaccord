# 프로젝트 소개 Introduction
'BlueAccord'는 모자 쇼핑몰 웹사이트입니다. 프로젝트 기간은 약 한달(6월 1일 ~ 7월 8일)간 진행되었으며, 개인적으로 진행한 프로젝트입니다.
백엔드 공부를 진행하면서 개발능력을 함양하고 싶었고, 그에 걸맞는 주제가 다양한 기능을 가진 쇼핑몰이라 판단하여 진행하게 되었습니다.
<br/><br/>
주요 기능으론 다음과 같습니다.
* 회원가입 및 로그인
* 제품 목록 및 검색
* 장바구니 및 주문 관리
* 관심상품
* 관리자 페이지 (현재는 제품관리만 구성)
* 마이 페이지
* 고객센터
* 리뷰 및 평점, 좋아요
